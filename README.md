# ⚽ Stadium Booking App

A modern and professional mobile application for booking football stadiums and sports fields, built using Flutter.  
The app provides a smooth and user-friendly experience for both customers and stadium owners.

## 🚀 About the App

This application was developed to simplify the process of searching, viewing, and booking sports stadiums.  
Users can browse available fields, check prices and available times, and make bookings بسهولة through an interactive and responsive interface.

The app also includes a dedicated dashboard for stadium owners to manage bookings, prices, and available schedules.

## ✨ Features

### 👤 User Features
- Browse stadiums by city
- Search across all stadiums
- View stadium details and images
- Book available time slots
- View upcoming and previous bookings
- Cancel bookings
- Dark mode support
- Smooth animations and modern UI

### 🏟️ Stadium Owner Features
- Manage stadium information
- Add, edit, and delete stadium images
- Update prices and available schedules
- Receive booking notifications
- Manage different stadium sizes and periods
